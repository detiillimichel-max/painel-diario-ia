# Assistente Pessoal Inteligente

Um assistente modular focado em produtividade com integração de pagamentos.

## Funcionalidades
- Interface moderna com Glassmorphism.
- Persistência de dados local.
- Navegação intuitiva.
- Integração segura com Mercado Pago.

## Como rodar localmente
1. Clone o repositório.
2. Abra o `index.html` no seu navegador.
3. Para funcionalidades de pagamento, utilize o ambiente de Sandbox do Mercado Pago.

## Deploy
Projeto configurado para deploy automático via Vercel.
